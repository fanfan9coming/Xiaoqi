# 扣子 Bot 配置指南

> 适用版本：2026-06-23
> 模型：豆包·1.5·pro
> 渠道：网页测试
> OCR：扣子自带
> 知识库：伤寒论条文（已建好）

---

## 一、整体架构

```
用户拍方 → OCR识别 → 工作流拼装 prompt
                ↓
        [知识库：伤寒论条文] ← 强制检索
                ↓
        豆包·1.5·pro 生成回答
                ↓
        用户看到：方解 + 条文引用
```

---

## 二、创建 Bot

扣子后台 → 资源库 → Bot → 创建 Bot
- 名称：中医方解助手
- 人设：见下方"人设与回复逻辑"
- 模型选择：豆包·1.5·pro（推荐 pro，常规版本也行）

---

## 三、绑定知识库

1. Bot 编辑页 → 知识库 → 添加
2. 选择已建好的「伤寒论条文」
3. **匹配设置**：
   - 召回方式：混合检索（向量 + 全文）
   - 召回数量：5-8 条
   - 最低匹配度：0.5
4. 保存

> 注意：金匮要略知识库待建（用户尚未提供 TXT）

---

## 四、配置工作流（OCR → 知识库 → 生成）

### 4.1 触发方式

设置 Bot 的输入方式为「图片 + 文字」双输入。

### 4.2 工作流节点（推荐顺序）

| 节点 | 类型 | 说明 |
|------|------|------|
| 1. 接收用户输入 | 开始节点 | 接图片和/或文字 |
| 2. OCR 识别 | 插件节点 | 用「图片内容理解」插件（扣子自带） |
| 3. 知识库检索 | 知识库节点 | 检索「伤寒论条文」 |
| 4. 拼装 prompt | 代码节点 | 见下方 prompt 模板 |
| 5. 调用 LLM | 大模型节点 | 豆包·1.5·pro |
| 6. 回复用户 | 结束节点 | 输出方解 + 条文引用 |

### 4.3 OCR 插件选择

扣子插件市场搜「图片内容理解」或「OCR」，选扣子官方提供的即可。

---

## 五、人设与回复逻辑（系统 Prompt）

复制以下内容到 Bot 的「人设与回复逻辑」：

```text
# 角色
你是一位严谨的中医方剂解读助手，专门帮用户解读《伤寒论》和《金匮要略》中的方剂。

# 工作方式
1. 你必须基于已检索到的「伤寒论条文」知识库内容回答
2. 每次回答必须引用至少 1 条相关条文（带条文编号，如 SHL-001）
3. 如果用户上传方剂图片，先 OCR 识别方名和药味，再检索相关条文
4. 如果知识库没找到相关内容，明确告诉用户"知识库中暂未收录该方"，不要编造

# 回答结构
对每个方剂，按以下结构回答：
【方名】原文方名（如：桂枝汤）
【出处】《伤寒论》第几条
【组成】药味 + 剂量（按原文）
【方解】简要方义分析
【条文原文】引用原始条文
【主治】主症（按原文）
【注意事项】该方禁忌或适用边界

# 限制
- 不做任何超出《伤寒论》《金匮要略》原文的临床推断
- 不替代专业医生诊疗，建议用户找专业中医师面诊
- 不回答与方剂解读无关的问题（礼貌拒绝并引导回主题）
```

---

## 六、强制引用规则（防止大模型瞎编）

在 Bot 的「开场白与建议问题」下方，有「回复逻辑 - 高级」配置，开启：

### 6.1 强制知识库引用

```text
你必须基于已检索到的知识库内容回答。
如果检索结果为空，明确告知用户"暂未收录该方"。
禁止编造条文、药味、剂量。
每次回答必须在结尾标注「引用条文：[条文ID列表]」。
```

### 6.2 免责声明（必须配置）

```text
本回答仅供参考，不构成医疗建议。
请在专业中医师指导下使用。
```

## 七、测试用例

### 7.1 上传图片测试

- 上传一张桂枝汤的方子（可手写可印刷）
- 验证 OCR 能否正确识别方名和药味
- 验证回答是否引用了 SHL-012、SHL-013 等条文

### 7.2 文字测试

- 直接输入"桂枝汤有什么组成"
- 验证回答是否引用了正确条文

### 7.3 边界测试

- 输入"麻黄汤的副作用" → 验证是否明确告知知识库未收录副作用
- 输入"今天天气" → 验证是否礼貌拒绝

---

## 八、注意事项

1. **首次配置后必须测试**：跑通"上传方子 → 看到引用条文"完整流程
2. **知识库召回率是关键**：如果发现漏检，降低最低匹配度（0.5 → 0.3）
3. **OCR 识别率**：手写方子识别率低，建议先测试印刷版
4. **免责声明必须保留**：医疗类输出必须有免责
5. **先单书测试**：先用「伤寒论条文」知识库跑通，验证后再加「金匮要略」
