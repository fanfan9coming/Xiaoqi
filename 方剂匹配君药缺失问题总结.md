# 方剂匹配算法缺陷：君药缺失导致误匹配

## 问题描述

在中医方解小程序的代码节点方剂匹配中，基于 IDF 加权的评分算法存在一个系统性缺陷：**对"大块头方剂"（药味众多的方剂）存在虚高评分**，导致匹配结果与临床实际偏离。

具体表现：当医生处方中并不包含某方的君药（核心主药）时，该方仍可能因药味数量多、命中数高而获得最高分，被错误地作为最终匹配结果输出。

## 典型案例

### 测试处方

```
当归 细辛 枳实 党参 赤芍 大枣 茯苓 白扁豆 白术 鸡血藤 
浮小麦 陈皮 桂枝 甘草 香附 薏苡仁 葛根 柴胡 白芍
```

（共 19 味不重复药，白术出现两次去重）

### 匹配结果（修复前）

| 来源库 | 匹配方剂 | Score | 命中数 | 缺失数 |
|--------|---------|-------|--------|--------|
| 金匮要略 | 薯蓣丸 | 22.371 | 9 | 12 |
| 方剂学 | 参苓白术散 | 21.147 | 6 | 4 |
| 伤寒论 | 当归四逆汤 | 16.711 | 6 | 1 |

### 问题分析

**薯蓣丸**得分最高（22.371），但该方由 21 味药组成（含薯蓣、人参、白术、茯苓、甘草、当归、白芍、地黄、阿胶、麦冬、五味子、杏仁、桔梗、豆黄卷、防风、柴胡、干姜、白蔹、桂枝、神曲、大枣），其**君药"薯蓣"（山药）根本不在医生处方中**。

一个连君药都没有的方剂，仅因为"块头大"、与处方有 9 味药的重叠（多为甘草、大枣、白术、茯苓、当归、白芍等常见药味），就被判定为最佳匹配——这在临床上是完全说不通的。

相比之下，**当归四逆汤**虽然得分最低（16.711），但其组成仅 7 味药（当归、桂枝、白芍、细辛、甘草、大枣、通草），处方命中了其中 6 味（除通草外全部命中），且**君药"当归"和"桂枝"均在处方中**，临床相关性远高于薯蓣丸。

## 根因

原评分公式：

```
score = Σ(IDF_i × hit_i) + 0.03 × hit_count - 0.3 × miss_count
```

该公式的问题在于：
1. **纯统计驱动，无结构性约束**：只考虑药味是否命中的 IDF 权重和命中/缺失数量，不考虑药味在方剂中的角色（君臣佐使）
2. **大方剂天然占优**：药味越多的方剂，与任何处方的"偶然重叠"概率越高，miss_count 的惩罚（0.3/味）不足以抵消命中带来的收益
3. **君药无特殊权重**：君药是方剂的核心，决定方剂的根本功效方向。君药缺失意味着该方的"灵魂"不在，理应被大幅降权甚至排除

## 解决方案：君药加权机制

### 算法

在原有 IDF 加权评分的基础上，引入君药命中率作为**乘性调节因子**：

```python
jun_yao = set(该方的君药集合)
jun_hit = 处方药味 ∩ jun_yao

if len(jun_hit) == 0:
    score *= 0.3    # 君药全未命中 → 重罚（降至 30%）
elif len(jun_hit) == len(jun_yao):
    score *= 1.2    # 君药全命中 → 奖励（提升 20%）
else:
    score *= 0.8    # 君药部分命中 → 轻度惩罚（降至 80%）
```

### 修复后效果

同样以薯蓣丸为例：
- 君药："薯蓣"（1 味）
- 处方中不含薯蓣 → `jun_hit = 0`
- 调节因子：×0.3
- 修正后 score ≈ 22.371 × 0.3 ≈ **6.71**

当归四逆汤：
- 君药："当归"、"桂枝"（2 味）
- 处方中含当归和桂枝 → `jun_hit = 2 = len(jun_yao)`
- 调节因子：×1.2
- 修正后 score ≈ 16.711 × 1.2 ≈ **20.05**

排序反转：当归四逆汤（20.05）> 参苓白术散 > 薯蓣丸（6.71），符合临床判断。

### 君药数据来源

君药数据基于《方剂学》教材、《伤寒论》《金匮要略》原文及方剂学权威参考书，为每个方剂标注君药集合（1~3 味），共建立 363 条君药映射记录，覆盖全部 355 个方剂（仅 1 条极冷门方缺失）。

## 经验总结

1. **纯统计匹配在中医药领域不够**：中医方剂有明确的结构性（君臣佐使），不能仅靠药味集合的重叠度来判断归属
2. **君药是方剂的"指纹"**：临床辨识一个方剂，首先看君药是否在处方中。没有君药，基本可以排除
3. **乘性调节优于加性补偿**：用乘法因子（×0.3 / ×1.2）而非加减常数来调节，能让君药信息在最终排序中起到决定性作用
4. **数据质量是关键**：君药数据的准确性直接决定了加权机制的效果，需要基于权威教材逐方核实
