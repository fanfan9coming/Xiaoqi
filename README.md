# Xiaoqi
小岐是中医智能助手，服务广大患者，可在中医问诊前后帮助用户理清自身病症，读懂医师辨证施治的诊疗逻辑，打通医患沟通壁垒，增进医患彼此信任。 Xiaoqi is an intelligent TCM assistant for patients. It helps users sort out symptoms and understand doctors’ syndrome differentiation and treatment ideas before and after TCM visits, smooth doctor-patient communication and boost mutual trust.
